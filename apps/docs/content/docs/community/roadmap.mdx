## Long-term Vision

### Framework Goals

1. **Performance**: Maintain 30,000+ RPS for simple apps
2. **Bundle size**: Keep core under 200KB
3. **Developer experience**: Zero-config where possible
4. **Type safety**: 100% TypeScript coverage
5. **Flexibility**: Support both functional and class-based patterns

### Non-Goals

- Replacing Express/Fastify for all use cases
- Building a full-stack framework (like Next.js)
- Supporting legacy Node.js versions (< 20)
