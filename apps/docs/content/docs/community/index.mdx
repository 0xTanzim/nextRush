## How to Contribute

- **Report a bug** — [Open an issue](https://github.com/0xTanzim/nextrush/issues/new) with reproduction steps and expected vs actual behavior.
- **Suggest a feature** — [Open an issue](https://github.com/0xTanzim/nextrush/issues/new) describing the use case and proposed behavior.
- **Improve documentation** — Fix errors, add examples, or clarify sections.
- **Submit code** — See the [Contributing Guide](/docs/community/contributing) for setup and PR process.

---

## Code of Conduct

All participants are expected to be respectful and inclusive. Harassment, discrimination, and personal attacks are not tolerated.
