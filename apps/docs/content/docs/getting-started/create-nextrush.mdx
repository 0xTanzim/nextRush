The `create-nextrush` scaffolder generates a ready-to-run NextRush project with routing, middleware, and optional DI/controllers ‚Äî no manual setup required.

## Quick Start

```bash
# Interactive (recommended)
pnpm create nextrush

# With npm
npm create nextrush

# With yarn
yarn create nextrush
```

The CLI walks you through project name, style, middleware preset, and runtime target.

## Non-Interactive

Pass flags to skip the prompts:

```bash
pnpm create nextrush my-api --style functional --middleware api --runtime node
```

## Project Styles

### `functional` (default)

Lightweight, function-based routing. Best for APIs and microservices.

```
my-api/
‚îú‚îÄ‚îÄ src/
‚îÇ   ‚îú‚îÄ‚îÄ index.ts
‚îÇ   ‚îî‚îÄ‚îÄ routes/
‚îÇ       ‚îî‚îÄ‚îÄ health.ts
‚îú‚îÄ‚îÄ package.json
‚îú‚îÄ‚îÄ tsconfig.json
‚îî‚îÄ‚îÄ .gitignore
```

```typescript title="src/index.ts"
import { createApp, createRouter, listen } from 'nextrush';

const app = createApp();
const router = createRouter();

router.get('/', (ctx) => ctx.json({ message: 'Welcome to NextRush!' }));

app.route('/', router);
await listen(app, 3000);
```

### `class-based`

Decorator-driven controllers with dependency injection. Best for structured applications.

```
my-api/
‚îú‚îÄ‚îÄ src/
‚îÇ   ‚îú‚îÄ‚îÄ index.ts
‚îÇ   ‚îú‚îÄ‚îÄ controllers/
‚îÇ   ‚îÇ   ‚îî‚îÄ‚îÄ health.controller.ts
‚îÇ   ‚îî‚îÄ‚îÄ services/
‚îÇ       ‚îî‚îÄ‚îÄ app.service.ts
‚îú‚îÄ‚îÄ package.json
‚îú‚îÄ‚îÄ tsconfig.json
‚îî‚îÄ‚îÄ .gitignore
```

```typescript title="src/controllers/health.controller.ts"
import { Controller, Get, Service } from 'nextrush/class';

@Service()
class AppService {
  getHealth() {
    return { status: 'ok', uptime: process.uptime() };
  }
}

@Controller('/health')
class HealthController {
  constructor(private readonly appService: AppService) {}

  @Get()
  check() {
    return this.appService.getHealth();
  }
}
```

## Middleware Presets

| Preset    | Included Packages                                                          |
| --------- | -------------------------------------------------------------------------- |
| `minimal` | None ‚Äî bare framework                                                      |
| `api`     | `cors`, `body-parser`, `helmet`                                            |
| `full`    | `cors`, `body-parser`, `helmet`, `rate-limit`, `compression`, `request-id` |

## After Scaffolding

```bash
cd my-api

# Start development server (hot reload)
pnpm dev

# Build for production
pnpm build

# Run production build
pnpm start
```

<Callout type="info" title="Code Generators">
  Once your project is set up, use `npx nextrush generate` to scaffold controllers, services,
  middleware, guards, and routes. See the [generators guide](/docs/guides/generators).
</Callout>
