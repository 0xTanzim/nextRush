This guide walks you through building a REST API with NextRush. You'll set up routing, middleware, and JSON responses.

<Callout type="info" title="Prerequisites">
  You need **Node.js 22.0.0+** and **pnpm** installed. TypeScript knowledge is recommended.
</Callout>

<Callout type="info" title="Scaffold a Project Instantly">
  Prefer an interactive scaffolder? Run `pnpm create nextrush` to generate a ready-to-run project
  with your choice of functional or class-based style, middleware preset, and runtime target. See
  the [create-nextrush guide](/docs/getting-started/create-nextrush) for details.
</Callout>

<Step>
### Create a new project

```bash
mkdir my-nextrush-api
cd my-nextrush-api
pnpm init
```

</Step>

<Step>
### Install dependencies

<PackageInstall packages={["nextrush"]} dev={["tsx", "typescript", "@types/node"]} />

<Step>
### Create the entry point

Create `src/index.ts`:

```typescript title="src/index.ts"
import { createApp, createRouter, listen } from 'nextrush';

// Create the application
const app = createApp();

// Create feature routers
const health = createRouter();
health.get('/', (ctx) => {
  ctx.json({
    name: 'My API',
    version: '1.0.0',
    status: 'healthy',
  });
});

const users = createRouter();
users.get('/', (ctx) => {
  ctx.json([
    { id: 1, name: 'Alice', email: 'alice@example.com' },
    { id: 2, name: 'Bob', email: 'bob@example.com' },
  ]);
});

users.get('/:id', (ctx) => {
  const { id } = ctx.params;
  ctx.json({
    id: Number(id),
    name: 'Alice',
    email: 'alice@example.com',
  });
});

// Mount routers
app.route('/', health);
app.route('/users', users);

// Start the server
await listen(app, 3000);
```

</Step>

<Step>
### Run the server

```bash
npx tsx src/index.ts
```

## Adding Middleware

Middleware runs before your route handlers. Add a request logger:

```typescript title="src/index.ts" {7-12}
import { createApp, createRouter, listen } from 'nextrush';

const app = createApp();

// Logger middleware
app.use(async (ctx) => {
  const start = Date.now();
  await ctx.next();
  const ms = Date.now() - start;
  console.log(`${ctx.method} ${ctx.path} - ${ms}ms`);
});

// Create and mount routers
const users = createRouter();
users.get('/', (ctx) => ctx.json([]));
app.route('/users', users);

await listen(app, 3000);
```

<Callout type="info" title="Middleware Order Matters">
  Middleware runs in registration order. The logger above runs **before** the routers because it was
  registered first with `app.use()`.
</Callout>

Now when you make requests, you'll see timing logs:

```
GET / - 2ms
GET /users - 1ms
GET /users/1 - 1ms
```

## Adding Built-in Middleware

NextRush provides production-ready middleware as separate packages. Here's how to add CORS and security headers:

```typescript title="src/index.ts" {1,5-6}
const app = createApp();

// Security middleware

app.use(cors());

// ... rest of your code
```

## Handling POST Requests

To accept JSON bodies, add the body parser:

<PackageInstall packages={['@nextrush/body-parser']} />

```typescript title="src/index.ts"
import { json } from '@nextrush/body-parser';

// Parse JSON bodies
app.use(json());

// Now you can handle POST requests
users.post('/', (ctx) => {
  const { name, email } = ctx.body as { name: string; email: string };

  const newUser = {
    id: Date.now(),
    name,
    email,
  };

  ctx.status = 201;
  ctx.json(newUser);
});
```

Test it:

```bash
curl -X POST http://localhost:3000/users \
  -H "Content-Type: application/json" \
  -d '{"name":"Charlie","email":"charlie@example.com"}'
```

## Complete Example

Here's the complete code with everything we've covered:

<details>
<summary>Full application code with routing, middleware, and POST handling</summary>

```typescript title="src/index.ts"
import { createApp, createRouter, listen } from 'nextrush';
import { cors } from '@nextrush/cors';
import { helmet } from '@nextrush/helmet';
import { json } from '@nextrush/body-parser';

const app = createApp();

// Middleware
app.use(helmet());
app.use(cors());
app.use(json());

// Logger
app.use(async (ctx) => {
  const start = Date.now();
  await ctx.next();
  console.log(`${ctx.method} ${ctx.path} - ${Date.now() - start}ms`);
});

// Feature routers
const health = createRouter();
health.get('/', (ctx) => {
  ctx.json({ name: 'My API', version: '1.0.0' });
});

const users = createRouter();
users.get('/', (ctx) => {
  ctx.json([
    { id: 1, name: 'Alice' },
    { id: 2, name: 'Bob' },
  ]);
});

users.get('/:id', (ctx) => {
  ctx.json({ id: Number(ctx.params.id), name: 'Alice' });
});

users.post('/', (ctx) => {
  const { name, email } = ctx.body as { name: string; email: string };
  ctx.status = 201;
  ctx.json({ id: Date.now(), name, email });
});

// Mount routers
app.route('/', health);
app.route('/users', users);

// Start
await listen(app, 3000);
```

</details>

## Common Mistakes

<Callout type="warn" title="Forgetting ctx.next()">
  If your middleware doesn't call `await ctx.next()`, the request will hang. Always call
  `ctx.next()` unless you're intentionally ending the response.
</Callout>

Middleware registered **after** `app.route()` calls won't run for those routes. Register middleware before mounting routers.
