## Learning Path

If you're new to NextRush, follow one of these paths based on your preferred style.

### Functional Style

| Step | Guide                                               | What You'll Learn                    |
| ---- | --------------------------------------------------- | ------------------------------------ |
| 1    | [REST APIs](/docs/guides/rest-api)                  | Project structure, routing, services |
| 2    | [Error Handling](/docs/guides/error-handling)       | HTTP errors, error middleware        |
| 3    | [Validation](/docs/guides/validation)               | Input validation with Zod            |
| 4    | [Custom Middleware](/docs/guides/custom-middleware) | Auth, logging, rate limiting         |
| 5    | [Testing](/docs/guides/testing)                     | Unit and integration tests           |

### Class-Based Style

| Step | Guide                                               | What You'll Learn             |
| ---- | --------------------------------------------------- | ----------------------------- |

| 5    | [Testing](/docs/guides/testing)                     | Unit and integration tests    |
