## Functional or Class-Based?

NextRush supports both functional and class-based styles. The [Hello World](/docs/examples/hello-world) and [REST API](/docs/examples/rest-api) examples use the functional style. The [Class-Based API](/docs/examples/class-based-api) example rebuilds the same REST API with decorators and dependency injection.

Start with functional if you're unsure — you can adopt class-based patterns incrementally.
