# Hello World

The simplest possible NextRush application.

---

## The Code

```typescript title="src/index.ts"
import { createApp, createRouter, listen } from 'nextrush';

const app = createApp();
const router = createRouter();

router.get('/', (ctx) => {
  ctx.json({ message: 'Hello, NextRush!' });
});

app.use(router.routes());

listen(app, 3000, () => {
  console.log('Server running at http://localhost:3000');
});
```

That's it. 10 lines to a working API.

## What Each Part Does

### createApp()

Creates the NextRush application instance. This manages middleware and plugins.

```typescript
const app = createApp();
```

### createRouter()

Creates a router for defining routes. NextRush uses a high-performance radix tree router.

```typescript
const router = createRouter();
```

### Route Definition

Define a GET route at the root path:

```typescript
router.get('/', (ctx) => {
  ctx.json({ message: 'Hello, NextRush!' });
});
```

- `ctx` — The context object containing request and response
- `ctx.json()` — Send a JSON response

### Mounting Routes

Register the router's routes with the application:

```typescript
app.use(router.routes());
```

### Starting the Server

Start listening for requests:

```typescript
listen(app, 3000);
```

## Adding More Routes

```typescript
import { createApp, createRouter, listen } from 'nextrush';

const app = createApp();
const router = createRouter();

// Root
router.get('/', (ctx) => {
  ctx.json({ message: 'Hello, NextRush!' });
});

// Health check
router.get('/health', (ctx) => {
  ctx.json({
    status: 'ok',
    uptime: process.uptime(),
  });
});

// Echo endpoint
router.get('/echo', (ctx) => {
  ctx.json({
    method: ctx.method,
    path: ctx.path,
    query: ctx.query,
  });
});

// Route with parameters
router.get('/users/:id', (ctx) => {
  ctx.json({ id: ctx.params.id });
});

app.use(router.routes());
listen(app, 3000);
```

## With Middleware

Add common middleware:

```typescript
import { createApp, createRouter, listen } from 'nextrush';
import { cors } from '@nextrush/cors';
import { json } from '@nextrush/body-parser';

const app = createApp();
const router = createRouter();

// Middleware
app.use(cors());
app.use(json());

// Routes
router.get('/', (ctx) => {
  ctx.json({ message: 'Hello, NextRush!' });
});

router.post('/echo', (ctx) => {
  ctx.json({ received: ctx.body });
});

app.use(router.routes());
listen(app, 3000);
```

<Callout type="info">
Install middleware packages separately: `npm install @nextrush/cors @nextrush/body-parser`
</Callout>
