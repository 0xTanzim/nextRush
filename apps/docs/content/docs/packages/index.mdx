# Packages

NextRush is organized as a modular monorepo. Install only what you need — core is under 3,000 lines of code.

## Core

| Package | Description | npm |
|---------|-------------|-----|
| [nextrush](https://www.npmjs.com/package/nextrush) | Core framework with routing, middleware, and HTTP server | `pnpm add nextrush` |
| [create-nextrush](https://www.npmjs.com/package/create-nextrush) | CLI scaffolding to create new projects | `pnpm create nextrush` |
